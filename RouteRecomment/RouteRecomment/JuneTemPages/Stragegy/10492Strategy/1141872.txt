

            新的一年开始了，作为迎接新一年的到来，决定在新年期间去一下不太遥远而且能适应冬天气温的地方 ,于是选择了咱们的隔壁省， 广西。众所周知，桂林山水甲天下，阳朔山水甲桂林。眼见为实，去到了才知道的确不是徒有虚名的。虽然那些天天气不太好，不过也没有妨碍我们新年之旅。  具体人物：我，我亲爱的表姐，表姐的朋友A和B  出发日期：2013年2月12日 回程日期：2013年2月15&amp; 2月16 旅程地点：桂林 + 阳朔  行程安排：Day 1（2月12日） 广州 -----（火车）----桂林 住：火车  今天开销：RMB 206 +1.5(水) + 16.5(晚) = 224       Day 2 （2月13日 )桂林一天游 - 两江四湖  住：桂林懒猫青年旅社  今日开销：4.5(早) + 7(午) + 3(香肠) + 37(晚) + 6.9(袜子)+ 60(住宿)= 118.4  下火车第一件事就是找东西吃了，来到桂林，当然要吃桂林米粉啦。由于我们下火车的时候是早上的6点半左右，很多店铺都还没有营业，我们边走边看，很幸运遇到一家吃的。我们当然不会放过了。      吃完我们的第一餐，然后就先去找住的地方把行李放下。桂林懒猫青年旅社不错，值得推荐。出发前就教了我们如何到达他们的旅店，而且交通比较方便。我们订了一个四人间，因为新年的关系，每个床位的价格变成了60/人，价格还可以接受，但是比较遗憾的是房间没有独立的洗手间。除了这意外，其他的一切还行。         放下行李后，我们就出去瞎逛了。两江四湖在这里我就不介绍了，百度多的是呢。                      逛着，聊着，买一下东西，很快天就黑了，本来这天就没有太阳，所以准确说应该是到肚子饿的时候了，打电话回旅店问有什么推荐的食店推荐，桂林粥城，很多人在等位啊.....基于多人吃就是好吃的关系，咱们等.......等的时间不长，上菜的时间也比较快！   桂林粥城出名的鸭子，味道不错。    田螺，像拇指长度的，比拇指宽一点点，味道一般。    这个是漓江小鱼，很辣，我们只迟了几条而已。    晚饭吃完，就继续我们的夜游两江四湖了，景色还真不错，除了天气有点冷之外，幸好没下雨。        夜游到8点多咱们就回去了，有点小插曲，就是咱们走到一半的时候找不到回去的路了，然后打电话回去旅店他们教了我们怎样回去，可是我们四个人都布确定怎样走，所以就唯有旅店的帅哥出来接我们回去了。我们问他，是不是第一次出来接找不到路的人回去，他说：不是啊，你们是我做这么久的第二个而已......那好吧，幸好不是他的第一次。哈哈。回到旅店咱们就赶紧洗澡睡觉准备明天的阳朔之旅了。 Day 3（2月14日）桂林 -- 杨堤 -- 兴坪 --阳朔 住：甜蜜之家  今日开销 1(车费) + 4.5(早) + 23(车费) + 5.5(小吃) + 7(车费) + 77(晚) + 60(住)=188  出发前就已经在淘宝定了由杨堤到兴坪的竹筏精华游，新年价钱是120/人，四个人一只伐，不够四人要补余下位置的价钱。咱们四个人就不需要补了，不过我觉得一个人120的确有点贵。 因为是预订了下午14:30份的船，所以这天我们可以睡到自然醒，我们先从旅店坐车到桂林火车站，然后根据攻略说的搭车去杨堤，其实在火车站外面就有很多阿姨问你要不要做竹筏之类的，价格貌似也比太贵，如果之前没有准备好的话可以考虑。从火车站去到杨堤大概坐了一个小时左右，价格25元 / 人。然后下车过对面马路再上到杨堤码头的车，3元 / 人。 去到码头后就打给我们的师父，后来来了个阿姨，原来她就是帮我们开船的，沿路都有讲解，而且这位阿姨说话还蛮风趣的 。可惜的是天公不作美啊。天气不尽大雾，而且还飘着雨，所以基本上沿江的风景都看不到。遗憾啊 。        九马画山，结果一只都看不到，不是我不聪明，是因为天气啦。    从杨堤码头到兴坪大概坐了两个小时，上岸也就是传说中20块钱的景点。一样，天气原因什么都看不到。上岸后大概走5分钟就到达了免费的电瓶车上车点，大家要记得把船票拿好啊，因为有船票才能免费坐电瓶车的。电瓶车大概坐了十多分钟就到了兴坪汽车站，然后再找阳朔的车去到阳朔。路程大概50分钟，车费7块 / 人。  到了阳朔后我们第一时间去汽车站预买车票回广州。之后就去找旅店了。旅店也是懒猫青年旅社，阳朔分店。但是由于他们员工在订房方面出错了，结果就安排我们呢四个人暂住隔壁甜蜜之家一晚，4个人两张大床，反正一晚而已，挤一挤就好了。至于这家甜蜜之家嘛，我觉得还蛮适合度假的情侣来住的，顾名思义。哈哈 -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- 从桂林到杨堤再到兴坪再到阳朔，这样就一天了，我们放下行李后大概8点多了，就去找吃的了，问了旅店哪个啤酒鱼好吃，推荐谢三姐。好，就去试一下。味道还行，其他菜也可以，值得推荐。  竹篙什么，名字忘记了，味道不错。       水煮牛肉跟清炒土豆干，味道可以。    传说中的啤酒鱼，我们这一条大概两斤半，4个人还吃不完了。味道不错。    吃完饭已经差不多10点了，咱们就回旅店休息，准备第二天的行程。 Day 4（2月15日）阳朔西街一天游 住：阳朔懒猫青年旅社  今日开销 6(早) + 2.5(小吃) + 65(晚) + 79(手信) + 60(住) = 212.5  本来原本计划是骑自行车逛十里画廊的，结果天气不给力，所以就只能在西街闲逛一天了。 阳朔的桂林米粉要比桂林的贵喔。而且手信这方面，我建议大家要货比三家再买，因为我们没有这样做，之后发现有些店铺卖的同样东西但是更便宜。  Day 5（2月16日）银子岩游   今日开销 23(午) + 68(门票) + 5(小吃) + 65(晚) + 250(车票) = 411  跟坐木筏一样，于出发前就在淘宝定了银子岩专线车+门票，68块 / 人。本人觉得银子岩值得去看一下的，虽然之前都已经看过其他的岩洞，但是本人觉得银子岩不错，可以游一下。而且不是有一句话吗：游了银子岩，一世不缺钱，哈哈。 从阳朔坐车到银子岩会经过十里画廊的，算是弥补骑不到车的遗憾吧。                我们去了没有饭店吃他们的没有菜，未到不错，可以看的景色也不错，至于价格嘛，肯定贵一点点啦。    传说中的香芋拔丝，其实蛮好吃的，不过很腻。     在阳朔我们既吃过了啤酒鱼，鸭，鸡，虾，最后不知道吃什么好了，就唯有吃PIZZA了，呵呵。    -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- 逛完了银子岩再坐专线车回到阳朔，再回旅店收拾东西，然后就出发去车站搭车回广州了。 四天三夜的旅程结束了。有机会的话还要来阳朔，这次的确有点遗憾。
